# pre-commit

https://pre-commit.com/

Using git hook

- missing semicolons,
- trailing whitespace,
- debug statements.

## Installation

```
brew install pre-commit
```

## Add a pre-commit configuration

```
.pre-commit-config.yaml
```


## show .git folder

.vscode/settings.json

```
"files.exclude": {
     "**/.git": false
}
```
